# bank-marketing-deposit

In this project, different machine learning models are applied, evaluated and compared on bank marketing data to predict whether a contacted customer will make a deposit. Random Forest, XGBoost, SVC and Logistic Regression are trained and compared, with the best models achieving an AUC of 0.74. This is significantly better than a random guess but in order to improve performance further information/ data is needed.

Data set: https://www.kaggle.com/datasets/nalisha/bank-marketing-dataset-for-deposit-prediction.
